# Sistema de Login em Python

Este é um projeto simples desenvolvido durante meus estudos de Python.

O objetivo do projeto é simular um sistema de login que permite até 3 tentativas de acesso antes de bloquear a conta.

## Funcionalidades

- Solicita usuário e senha
- Valida as credenciais digitadas
- Permite até 3 tentativas de login
- Bloqueia o acesso após exceder o limite de tentativas

## Tecnologias utilizadas

- Python

## Conceitos praticados

Neste projeto pratiquei conceitos fundamentais da programação:

- Estruturas condicionais (if / else)
- Estruturas de repetição (while)
- Contagem de tentativas
- Comparação de valores
- Lógica de programação

## Como executar

1. Clone este repositório
2. Abra o arquivo `login.py`
3. Execute o programa com Python
